<img width="886" height="593" alt="image" src="https://github.com/user-attachments/assets/35d9323d-acc8-4358-809c-3591a11f87e7" /># Платформа онлайн-обучения (WPF)

Приложение для управления курсами, студентами и учебным процессом с системой прогресса, оценок и сертификатов.

## Описание проекта
Система включает:
1. Каталог курсов с возможностью записи
2. Личный кабинет студента с пройденными курсами
3. Интерактивные уроки с текстовыми лекциями
4. Задания двух типов: практические и контрольные работы
5. Автоматический расчёт прогресса и итоговых оценок
6. Формирование сертификатов о прохождении курса

Архитектура построена на паттерне MVVM с использованием WPF и PostgreSQL. Бизнес-логика вынесена в отдельный модуль Core с полным покрытием unit-тестами.

## Возможности
- ✅ Запись на курсы из каталога
- ✅ Просмотр лекций с контролем прокрутки текста
- ✅ Выполнение практических и контрольных заданий
- ✅ Гибкая система оценивания (2-5 с конвертацией в баллы)
- ✅ Отслеживание прогресса по курсу в реальном времени
- ✅ Журнал оценок с детализацией по заданиям
- ✅ Генерация текстового сертификата
- ✅ Модульные тесты бизнес-логики (xUnit)
- ✅ XML-документация всех публичных элементов

## Технологии
- C# (.NET 8.0)
- WPF (XAML, MVVM)
- PostgreSQL 14+ (Npgsql)
- xUnit (модульное тестирование)
- Git (контроль версий)

## Запуск
1. Установите PostgreSQL и создайте базу данных `learning_platform`
2. Выполните SQL-скрипт создания таблиц (файл `docs/database_schema.sql` в pgAdmin)
3. Откройте файл `OnlineLearningPlatform.sln` в Visual Studio
4. В файле `App/Data.cs` укажите ваш пароль от PostgreSQL (строка подключения)
5. Соберите проект в режиме Release: **Сборка → Перестроить решение**
6. Запустите `App/bin/Release/net8.0-windows/App.exe`

## Структура проекта
- `App/` – WPF-приложение (интерфейс, работа с БД)
- `Core/` – бизнес-логика (сущности, сервисы расчётов, XML-документация)
- `Tests/` – модульные тесты для Core
- `docs/` – SQL-скрипты и документация
- `bin/Release/` – исполняемый файл для запуска
- `.gitignore` – настройки исключения файлов
- `.gitattributes` – настройки кодировки UTF-8

## Требования к окружению
- Windows 10/11
- .NET 8.0 Runtime (или SDK для разработки)
- PostgreSQL 14 или новее
